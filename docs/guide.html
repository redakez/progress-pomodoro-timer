Timer Guide About

Progress Pomodoro Timer uses pomodoro technique while keeping track of completed pomodoros. You can set a daily goal in the "Settings" tab, where the number of pomodoros and their length can be set.

The progress is always visible in the top part of the application. Then, there are four tabs with different functions. The timer tab is used to start, pause and end pomodoros. Manager tab can be used to remove or add progress, for instance when you forgot to start the pomodoro. The settings tab is used to specify the number of pomodoros and their length. Finally, the Logs tab contains a table of all actions since the last page load. There is also a graph which shows how the progress changes in time.
